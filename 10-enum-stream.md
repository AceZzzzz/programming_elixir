## 10.1-枚举类型
Elixir提供了枚举类型（enumerables）的概念，使用[Enum模块]()操作它们。我们已经介绍过两种枚举类型：列表和图。
```
iex> Enum.map([1, 2, 3], fn x -> x * 2 end)
[2, 4, 6]
iex> Enum.map(%{1 => 2, 3 => 4}, fn {k, v} -> k * v end)
[2, 12]
```

Enum模块为枚举类型提供了大量函数来变化，排序，分组，过滤和读取元素。
